# Mercury 用户流程导航图

## 📋 概述

本文档展示 Mercury RSS 阅读器的完整用户流程和页面间导航关系。

---

## 🗺️ 核心流程图

```
主界面 (main-interface-complete.html)
├── 标题栏
│   ├── [导入 OPML] → opml-import-dialog.html
│   └── [设置] → settings-page-complete.html
│
├── 左侧边栏
│   ├── [+ 添加订阅] → add-subscription-dialog.html
│   ├── [刷新] → 刷新所有订阅源
│   └── Feed 列表项
│     └── [⚙️ 编辑] → edit-subscription-dialog.html
│
├── 中间文章列表
│   ├── [全部/未读/已读/收藏] → 筛选文章
│   └── 文章项 → 点击加载到右侧阅读区
│
└── 右侧阅读区
    ├── [🏷️ 添加标签] → 标签选择对话框（待设计）
    ├── [📤 导出] → 导出选项菜单（待设计）
    ├── [🔗 原文链接] → 在浏览器打开
    └── [⚙️ 阅读设置] → reading-style-panel.html

设置页面 (settings-page-complete.html)
└── [← 返回主界面] → main-interface-complete.html
```

## 🎯 用户核心任务流程

### 任务 1：添加第一个订阅源

```
1. 启动应用 → 看到"无订阅源"空状态
2. 点击"+ 添加订阅源"按钮
3. 在对话框中输入 RSS URL
4. 点击"添加订阅"
5. 返回主界面，左侧显示新订阅源
6. 自动刷新并加载文章
```

**流程完整性：** ✅ 完整

---

### 任务 2：从其他阅读器迁移

```
1. 点击标题栏"导入 OPML"按钮
2. 选择或拖拽 OPML 文件
3. 预览订阅源列表，全选或单选
4. 点击"导入选中的订阅源"
5. 显示导入进度
6. 完成后返回主界面
```

**流程完整性：** ✅ 完整

---

### 任务 3：阅读文章并使用 AI 功能

```
1. 在左侧选择订阅源
2. 在中间列表选择文章
3. 右侧显示文章内容
4. 点击"生成摘要"按钮
5. 等待 AI 生成（显示加载状态）
6. 查看摘要结果
7. 可选：点击"生成翻译"
```

**流程完整性：** ✅ 完整

---

### 任务 4：配置 AI 服务

```
1. 点击标题栏"设置"按钮
2. 默认进入"AI 配置"页面
3. 填写 API Base URL 和 API Key
4. 点击"测试连接"验证
5. 点击"保存设置"
6. 点击"← 返回主界面"
```

**流程完整性：** ✅ 完整

---

### 任务 5：调整阅读样式

```
1. 在阅读区点击"⚙️ 阅读设置"按钮
2. 打开阅读样式面板
3. 调整字体、字号、行距、主题
4. 点击"×"关闭面板
5. 设置立即生效
```

**流程完整性：** ✅ 完整

---

### 任务 6：管理订阅源

```
1. 在左侧 Feed 列表悬停某个订阅源
2. 点击"⚙️"按钮
3. 打开编辑订阅对话框
4. 修改自定义名称或刷新频率
5. 可选：点击"删除订阅源"（危险操作）
6. 点击"保存修改"或"取消"
```

**流程完整性：** ✅ 完整
