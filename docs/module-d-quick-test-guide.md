## 常见问题

### Q: 点击"添加标签"没反应？
**A:** 检查浏览器控制台是否有错误。确保：
- 已选中一篇文章
- 主进程已重新编译（npm run build:main）

### Q: 导出按钮点击后没弹出对话框？
**A:** 确保：
- Electron 进程有权限访问文件系统
- 已选中一篇文章

### Q: 设置无法保存？
**A:** 检查：
- SQLite 数据库权限
- 控制台是否有错误信息

## 开发者检查清单

如果功能不工作，按顺序检查：

1. ✅ **编译主进程**
   ```bash
   cd D:/project/Mercury
   npm run build:main
   # 应该无错误
   ```

2. ✅ **检查 Vite 运行**
   ```bash
   curl http://localhost:5173
   # 应该返回 HTML
   ```

3. ✅ **检查 Electron 进程**
   ```bash
   tasklist | grep electron
   # 应该看到 5 个进程
   ```

4. ✅ **查看控制台错误**
   - 在 Electron 窗口中按 `Ctrl+Shift+I` 打开开发者工具
   - 查看 Console 标签是否有错误

5. ✅ **验证数据库**
   ```bash
   # 检查 settings 表
   sqlite3 ~/AppData/Local/mercury/mercury.db "SELECT * FROM settings;"
   
   # 检查 tags 表
   sqlite3 ~/AppData/Local/mercury/mercury.db "SELECT * FROM tags;"
   ```
